### Masala 1

##### Description

> Berilgan butun son `N` ni o'qing va `N` ning faktorialini hisoblang. 

> `factorial()` funksiyasi:
> - Argumentlari: bitta butun son `N`.
> - Return type: `int`.
> - `N` ning faktorialini qaytaradi.

#### Input

Bitta butun son `N` (`0 ≤ N ≤ 12`).

#### Output

`N` ning faktorialini chop eting.

#### Namuna 1
| Input | Output |
| - | - | 
| `5`  | `120` |
